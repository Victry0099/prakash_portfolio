News Website fetch real time news

Welcome to Real News, the ultimate source of news from around the world. Real News is a web application that uses React, a popular JavaScript library for building user interfaces, to create a fast and responsive news website.

With Real News, you can:

- Browse the latest headlines from various categories, such as business, entertainment, health, science, sports, and technology.
- Search for news articles by keywords, sources, or domains.
- Read the full article on the original website with a single click.
- Share your favorite articles with your friends on social media platforms.

Real News uses the Bing News Search API, a powerful and intelligent service that provides relevant and comprehensive news results. The API uses artificial intelligence to ensure the most accurate and up-to-date news articles.

Real News is built with React class-based components, which are reusable pieces of code that manage their own state and lifecycle. Class-based components inherit from the React.Component class and have access to its methods and properties. Class-based components also have a render() method, which returns the HTML to be displayed on the web page.

Real News consists of the following class-based components:

- App: The main component that renders the entire application and handles the state and logic of the news search and display.
- Header: The component that renders the header section of the web page, which contains the logo, the navigation menu, and the search bar.
- NewsList: The component that renders the list of news articles, which are fetched from the API using the query parameters from the state of the App component.
- NewsItem: The component that renders a single news article, which contains the title, the source, the date, the image, and the link to the original website.
- Footer: The component that renders the footer section of the web page, which contains the social media icons and the credits.

Real News is a simple and elegant news website that showcases the power and flexibility of React and the Bing News Search API. You can check out the source code of Real News on [GitHub]

Weather Website fetch real time using API

Do you want to know the current weather conditions and forecast for any location in the world? If so, you will love our weather website, which uses the OpenWeatherMap API and JavaScript to provide you with reliable and up-to-date weather information.

Our weather website is simple and easy to use. All you need to do is enter the name of the city, region, or country that you are interested in, and click the search button. You will then see the current weather data, such as temperature, humidity, wind speed, and pressure, as well as a 5-day forecast with daily and hourly details.

Our weather website is powered by the OpenWeatherMap API, a free and open-source service that offers global weather data for more than 200,000 locations. The API uses artificial intelligence and machine learning to ensure the accuracy and timeliness of the weather data.

Our weather website is built with JavaScript, a popular and versatile programming language that runs in the browser. We use the Fetch API, a modern and easy way to make HTTP requests and handle responses, to communicate with the OpenWeatherMap API. We also use HTML and CSS to create the layout and style of the web page.
